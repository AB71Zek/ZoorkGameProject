Commands in ZOOrk Game:
----------------------
go <direction> - Move in the specified direction (go north, go east, go south, go west, go down, go up).
look - Display the message "You don't see anything here."
look <object> - Display a message indicating that the object isn't found.
take/get - (Not yet implemented) Will eventually allow you to pick up items.
drop - (Not yet implemented) Will eventually allow you to drop items.
quit - Exit the game.
